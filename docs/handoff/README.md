## 🎯 Quick Project Summary

* **Project Title:** Kizuna (絆) — The Purposeful Connection Platform
* **Theme:** Tsunagaranai Kachi (つネットがらない価値) — Disconnected by Design
* **Primary Focus:** Moving from "broad & shallow" megaplatform bloat to "narrow & deep" human bonds.
* **Core Feature Suite:**
  1. **Sanctuary:** 15-friend inner circle cap & flexible delivery cadence (3x/wk, 5x/wk, Sunday Digest).
  2. **Omoide (思い出):** 1-on-1 private memory canvas, voice notes, and annual "Kizuna Wrapped" relationship reflections.
  3. **Fireside:** Micro-group living room circles (3–5 friends) with weekly reflective prompts and simultaneous answer reveals.
  4. **Shizuka (静か):** Solitude focus shield with zero red badges, read receipts, or intrusive popup banners.
