# 🌸 Kizuna (絆)
### Redesigning Connections: Moving from "Broad & Shallow" Megaplatforms to "Narrow & Deep" Intimacy

**OGIS Contest 2026 Proposal Presentation**  
*Theme: Tsunagaranai Kachi (つながらない価値) — Disconnected by Design*  
**Team Antigravity**

---

## 1. Dual Framework: Interpersonal & Intrapersonal

```
    INTERPERSONAL CONNECTIONS                    INTRAPERSONAL SOLITUDE
  ┌─────────────────────────────────┐        ┌──────────────────────────────┐
  │ • Sanctuary 15-Friend Cap       │        │ • Shizuka Solitude Sanctuary │
  │ • Omoide Locked Time Capsules   │   ───  │ • Private Self-Reflection    │
  │ • Fireside Living Room Circles  │        │ • Zero Dopamine Dark Mode    │
  └─────────────────────────────────┘        └──────────────────────────────┘
```

* **Interpersonal (With Close Friends):** Slow, intentional communication built around shared memory scrapbooks, locked time capsules, and living rooms.
* **Intrapersonal (With Oneself):** Quiet reflection and solitude space protecting personal mental balance.

---

## 2. Core Feature 1: Sanctuary 🏰

### Inner Circle Boundary & Flexible Cadence

* **15-Friend Cap (Dunbar Boundary):** Grounded in evolutionary psychology, users curate a living circle of maximum 15 close friends/family to enforce quality over quantity.
* **Stress-Free Delivery Cadence:** Delivery is **never forced daily**. Users set their preferred pacing:
  * **3 Days / Week** (Mon / Wed / Fri)
  * **5 Days / Week** (Weekdays)
  * **Sunday Digest** (Weekly quiet bundle)
* **Fading Seasons Archive:** Inactive contacts naturally drift into a peaceful memory archive without awkward un-friending.

---

## 3. Core Feature 2: Omoide (思い出) 🌸

### Interpersonal 1-on-1 Canvas & Locked Time Capsules

* **Interpersonal Time Capsules:** Friends can co-create digital Time Capsules containing letters, photo memories, and 60-second voice journals that remain **locked until a future date** (e.g. 1 year later, graduation, reunion).
* **Private Shared Timeline:** A 2-person visual scrapbook recording real shared history over time with **zero public likes or clout metrics**.
* **Kizuna Wrapped:** Inspired by Spotify Wrapped, Omoide automatically compiles a private, co-created yearly relationship reflection between two friends.

---

## 4. Core Feature 3: Fireside 🔥

### Micro-Group Living Room Circles

* **Micro-Groups (3–5 Close Friends):** Private rooms for tight-knit groups away from public feeds and noisy group chats.
* **Weekly Reflective Questions:** Prompts one deep question per week (e.g., *"What is something you've been quietly carrying this month?"*).
* **The Unveil (Simultaneous Reveal):** All members write privately; answers unlock for the circle only when all members submit, fostering genuine vulnerability without instant-reply anxiety.

---

## 5. Core Feature 4: Shizuka (静か) 🛡️

### Intrapersonal Solitude Shield & Full Dark Mode

* **Intrapersonal Self-Reflection Journal:** A private sanctuary for the user's own mind, providing quiet mood check-ins and personal journaling away from social performance.
* **Zero Dopamine Triggers:** Eliminates red notification badges, read receipts, typing indicators, and intrusive popup banners.
* **Full Night Mode:** Warm dark paper palette (`#181716` background, soft sepia text, warm gold borders) for nighttime solitude.

---

## 6. Design Lens: Neocities Retro-Personal Aesthetics

Inspired by personal web corners (e.g., *Stimmie.dev v3 Neocities Revival*):

* **Scrapbook Styling:** Taped photo frames, paper texture backgrounds, warm terracottas and soft cream tones.
* **Non-Corporate Typography:** Friendly display headers (*Fredoka*) and rounded sans-serif body (*Nunito / Quicksand*).
* **Organic Layout Offsets:** Elements feature subtle paper offsets and warm borders for a human, non-AI corporate touch.
